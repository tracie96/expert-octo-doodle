### Frontend

1. Navigate to the frontend directory:

2. clone repository:

    ```sh
    git clone https://github.com/tracie96/expert-octo-doodle.git 
    ```

    ```sh
    cd todo_fe
    ```

2. Install dependencies:

    ```sh
    yarn install
    ```

3. Start the React app:

    ```sh
    yarn start
    ```
